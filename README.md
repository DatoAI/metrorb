**MET**rics er**RO**r for **R**u**B**y

Metrorb is a collection of error metrics.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'metrorb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install metrorb
